InfLLM: Training-Free Long-Context Extrapolation for LLMs with an Efficient Context Memory

Released Date : 2024 02 07

Review Date : 2024 10 04

token을 split하여 KV cache를 줄임

Motivation#

  • KV Cache는 효과적인 attention을 위한 방법이다. 하지만 Long Tokens의 경우 많은 Cache가 필요하기에 메모리를 많이 차지하는 문제가 존재한다. 이번논문에서는 해당 문제를 Token Split으로 해결할려고 한다.

Methodology#

  • Token들을 특정 사이즈로 짜른다. 초기 토근(0,l), 중간토근(l,nl), Local token(nl,nl+l)을 이용하여 Evicted Token을 정한다. Evicted tokens는 Context Memory에 저장되며 실제 Gpu에는 Representative Tokens(r_m)만 저장된다. Current token과 r_m을 내적을 구하여 가장 높은 점수의 k를 가져와 Initial, Evicted, Local Token들로 attention을 수행한다.
